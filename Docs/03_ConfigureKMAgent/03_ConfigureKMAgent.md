# Exercise 03: Configure the Knowledge Management Agent


### Introduction

Contoso's current article creation process is too complex and inconsistent to scale-especially during busy periods when customer questions spike. In this exercise, you'll configure the Knowledge Management Agent so it can create knowledge from resolved cases (in real time and from historical cases), apply rules that control when articles should be created, and define publishing governance so new content improves consistency without introducing inaccurate or unreviewed guidance.

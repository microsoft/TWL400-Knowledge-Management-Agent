# Exercise 01: Set up and enable necessary connections and flows


### Introduction
Contoso's knowledge challenges aren't only about content quality-they're also about reliability. With articles spread across different sources and no consistent process, even strong content can't help if the automation that creates and updates the content isn't connected and running. In this exercise, you'll prepare the foundation that lets Contoso's Knowledge Management Agent consistently harvest case details and generate draft articles without adding extra manual effort for representatives.

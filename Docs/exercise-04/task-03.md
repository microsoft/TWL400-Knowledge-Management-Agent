# Task 03: Demonstrate Knowledge Management agent capabilities

Now that you have the case open and have walked the customer through the case creation process, you'll use Knowledge Management to assist in resolving the case. (This assumes that you've completed the Case Management Lab.)

## Steps

1. If necessary, select the Copilot icon on the right.

    {: .highlight }
    > **Hint:** Call out to the customer how Copilot helps to streamline case resolution with AI assistance to help identify potential solutions.

1. After talking through a resolution as part of the demo, add a **Note** to the case that includes the following:

    ```
    The machine needed to be recalibrated. To recalibrate the machine, I did the following:

    - Had the customer put the machine into service mode by holding the button on the back of the machine.

    - Logged into the device using the secure code provided by the customer.

    - Selected the recalibrate command.

    - I waited about 10 minutes for the machine to recalibrate.

    - Once the machine was recalibrated, I sent a remote reset command to the device.

    - Logged out of the device.
    ```

1. Next, walk through the process of resolving the issue by selecting **Resolve Case** in the **Case** tab.

1. Select **Problem Solved** as the category and enter a resolution detail of `Knowledge Article`.

    {: .highlight }
    > **Hint:** Since you're showcasing the automated ability, you'll want to clear the option to suggest article. That is manual.

1. Select **Resolve**.

1. Once the case is resolved, go to the case **Timeline**.

1. Select the **Suggested actions** tab.

    {: .highlight }
    > **Hint:** This will only appear when there's a suggested action.

1. Select **Review**.

1. In the **Review knowledge draft (preview)** dialog that appears, you can review and make any necessary edits.

1. Optionally, you can use Copilot to automatically edit the article by entering instructions in **Revise with instructions**, and then select **Re-generate**. For example, you can prompt Copilot to make the content more concise, and then select **Re-generate**.

1. Select **Mark as reviewed**. If you have the required permissions, you can select **Split and publish the article**.

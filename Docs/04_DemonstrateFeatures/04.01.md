## Task 01: Demonstrate self-service capabilities


You want to be able to answer questions related to warranty claims. To do this you'll expose warranty information to a self-service agent. To collect that data, you will need to first create a warranty table in Dynamics 365 to store the details.

1. In Edge, go to https://make.powerapps.com.

1. Go to **Apps**.

1. Locate and select the **Contoso Self Service** portal that you created in the prerequisites lab.

1. Verify that your chat widget displays the portal.

	{: .note }
    > If you're still not seeing the chat widget in the portal, look at the working hours associated with the chat to verify that you're within the defined work hours.


1. Select the Chat widget and enter to talk to an agent.

	{: .highlight }
    >  If you do not see the chat widget in the portal, look at the working hours associated with the chat to verify that you're within the defined work hours.

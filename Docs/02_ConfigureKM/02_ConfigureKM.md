# Exercise 02: Configure Knowledge Management functionality and tailor the environment


### Introduction
Contoso needs a knowledge system that produces consistent answers, preserves expertise as senior reps leave, and reduces the friction of creating articles. In this exercise, you'll configure the baseline Knowledge Management settings and Copilot behavior so Dynamics 365 can use Contoso's internal knowledge base as the source of truth and support faster, more standardized article creation.
